# checks disable

Manually disable a health check.

The check stops running but the rule is preserved. Re-enable with
'strada checks enable `<id>`'.

## Usage

```sh
strada checks disable <id>
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<id>` | Yes | id |
