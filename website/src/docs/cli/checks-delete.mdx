# checks delete

Delete a health check by ID.

Removes the check rule from D1. Historical check results remain in
ClickHouse and are queryable via 'strada query'.

## Usage

```sh
strada checks delete <id>
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<id>` | Yes | id |
