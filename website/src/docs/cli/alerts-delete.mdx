# alerts delete

Delete an error alert rule.

Destinations are not deleted; they remain for other rules.
Get the rule ID from 'strada alerts list'.

## Usage

```sh
strada alerts delete <id>
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<id>` | Yes | id |
